# Tsetlin Machine

> The Iris Dataset

## Quick Start

Using `pip`:
```
$ python -m pip install -r requirements.txt
```

Using `pipenv`:
```
$ python -m pip install pipenv
$ pipenv install
$ pipenv shell
```

```
$ python main.py
```
